Automated Network Alert
This project automates the management of Cisco IOS-XE devices using Python.

Key Features:
Auto-Backup: Pulls running-configs and saves them with timestamps to an AWS server.
Health Monitoring: Audits interface statuses in real-time.
Discord Integration: Sends instant alerts to a Discord channel via Webhooks.

Tools Used:
Python / Netmiko (SSH Automation)
AWS CloudShell / EC2 (Cloud Infrastructure)
Discord API (Observability & Alerting)
